# PROJECT_CONSTITUTION

Document ID: PRJ-CONST-001
Version: 0.1
Status: Accepted

# Purpose

This document defines the non-negotiable principles of the Wealth Platform.

Every product decision, architectural decision, implementation and AI-generated contribution must comply with this constitution.

# Product Mission

Help people understand, manage and plan their wealth by transforming fragmented financial information into trustworthy, explainable and actionable knowledge.

The platform supports decisions.
The user always makes the final decision.

# Constitutional Principles

P1 User Ownership
P2 Trust Before Features
P3 Financial Accuracy
P4 Financial Engine is the only calculation engine
P5 AI Never Calculates Finance
P6 Explainability
P7 Portfolio-Centric Thinking
P8 Human Decision
P9 Automation First
P10 Domain Before Technology
P11 Modular Architecture
P12 Complete Traceability
P13 Historical Integrity
P14 Security by Design
P15 Privacy by Design
P16 Explainable Recommendations
P17 Documentation Driven Development
P18 AI Ready Documentation
P19 Long-Term Maintainability
P20 Continuous Evolution
P21 Financial Transparency

# Amendment Rule

Any modification requires review and an approved ADR.
