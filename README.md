# Wealth Platform

The Wealth Platform is a Wealth Intelligence Platform designed to help individuals and professional wealth managers represent, understand and plan wealth.

This repository contains the documentation, architecture and future implementation of the platform.

## Repository Structure

```text
docs/        Product, domain and architecture documentation
adr/         Architecture Decision Records
templates/   Document templates
.cursor/     Cursor rules, prompts and project context
scripts/     Utility scripts
```

## Current Phase

Domain and architecture documentation.

## Core Domain Ideas

- Wealth is structured, not a single number.
- Position is the atomic unit of owned wealth.
- Assets are canonical master data.
- Transactions are immutable.
- Portfolios group Positions.
- Financial Plans define the planning context.
- AI explains and assists but does not define financial truth.
