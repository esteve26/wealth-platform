# PROJECT_CONTEXT

Project: Wealth Platform  
Purpose: Build a Wealth Intelligence Platform for individuals, wealth managers, financial advisors, family offices and internal administrators.

The platform is designed to represent, analyse and plan wealth using a coherent domain model.

## Product Identity

The Wealth Platform is not only a portfolio tracker.

It is a platform for:

- representing a user's wealth;
- understanding how wealth changes;
- organizing wealth around financial plans;
- supporting decisions through deterministic financial calculations;
- explaining information through AI.

## Current Domain Foundation

The project currently includes these core concepts:

- Workspace
- Wealth
- Financial Plan
- Portfolio
- Position
- Holding
- Asset
- Container
- Provider
- Transaction
- Goal
- Strategy
- Milestone
- Financial Engine
- AI Assistant

## Working Principle

Before implementing anything, the domain must be documented clearly enough that an AI coding agent can implement it with minimal ambiguity.
